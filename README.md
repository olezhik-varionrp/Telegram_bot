# 📋 Telegram Bot для заявок

Бот принимает заявки от пользователей и уведомляет администратора.

## Что умеет бот

- Пользователь заполняет заявку пошагово
- Администратор получает уведомление с кнопками Принять / Отклонить
- Пользователь получает уведомление о решении
- Все заявки сохраняются в `applications.json`

## Поля заявки

- Название канала
- Ссылка на канал
- Реальное имя
- Возраст
- Стаж на проекте
- Рассказ о себе
- Discord
- Telegram

## Установка

```bash
git clone https://github.com/твой_ник/твой_репозиторий
cd твой_репозиторий
npm install
cp .env.example .env
```

Заполни `.env` файл:
```
BOT_TOKEN=токен_от_BotFather
ADMIN_ID=твой_telegram_id
```

## Запуск

```bash
# Обычный запуск
node bot.js

# Через PM2 (рекомендуется на сервере)
pm2 start bot.js --name mybot
pm2 save
```

## Команды

| Команда | Описание |
|---|---|
| `/start` | Начало работы |
| `/cancel` | Отменить заявку |
| `/applications` | Список заявок (только для админа) |
